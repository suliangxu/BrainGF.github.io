什么是机器学习

学习（获得技巧）：从观察出发，经过脑子的转化，变成一个技巧。

![image-20200212223353499](/Users/hhh/Library/Application Support/typora-user-images/image-20200212223353499.png)

机器学习：从数据的积累来获得技巧

![image-20200212223544399](/Users/hhh/Library/Application Support/typora-user-images/image-20200212223544399.png)

什么是技巧？

某种表现的增进。（如预测的准确性）

机器学习：

从数据出发，经过电脑的计算之后，最终得到某一种表现的增进。

Machin Learning:

improving some preformace measure with experience computed from data.

![image-20200212223753894](/Users/hhh/Library/Application Support/typora-user-images/image-20200212223753894.png)

为什么使用机器学习？

机器学习的应用：

  • 图像识别

树的识别：

1、树的定义使用程序很难实现

2、从数据（观察）学习，然后进行识别

3、让机器来学习，可能会比写出若干行的程序来说明如何识别一个树，来的更简单一些

机器学习：让机器自己来学会、分析数据，来获得某种能力。

（ML: an alternative route to build complicated systems）

当人们在程序无法表达或形象的实现某个功能或无法无法制定规则时，机器学习不失为一种较好的选择。

  1. 操纵机器人在火星的运行
  2. 图像识别
  3. 做预测
  4. 个人化服务

Give a computer a fish, you feed it for a day; teach it how to fish, you need it for a lifetime.

机器学习的三个关键：（判断是否适合使用机器学习）

1、存在潜藏的模式（有某些目标）

如股市的目标：更准确。可能有某一种可以学的规则，机器可以从中学到。

2、不知道如何将这个规则写下来

3、有数据
