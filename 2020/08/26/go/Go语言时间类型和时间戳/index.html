二、时间戳

时间戳：一个 int64 的整数

表示意义：格林威治时间1970年01月01日00时00分00秒(北京时间1970年01月01日08时00分00秒)起到此时此刻的总秒数

函数：time.Now( ).Unix( )

运行结果：

注：

  • 此处加不加 Local()、UTC() 结果都是一样的
  • time.Now().UnixNano() 返回的是纳秒数
