一、psutil模块的介绍

在Python中，我们可以使用psutil这个第三方模块去获取信息的信息。

psutil模块可以跨平台使用，支持Linux／UNIX／OSX／Windows等，它主要用来做系统监控，性能分析，进程管理等。

安装psutil模块也非常简单，在cmd命令行下输入：pip install psutil

二、psutil模块的使用

1、获取CPU信息

psutil.cpu_times( )

获取CPU的完整信息。

psutil.cpu_count( )

获取CPU的逻辑个数。

psutil.cpu_count(logical=False)

获取CPU的物理个数；默认logical值为True

cpu_percent( )

获取系统CPU使用率。

其有两个参数，分别是interval和percpu；

interval指定的是计算cpu使用率的时间间隔，percpu则指定是选择总的使用率还是每个cpu的使用率；

2、获取内存信息

psutil.virtual_memory( )

获取系统内存的使用情况。

可以看到，总内存大小是8457662464KB=8 GB，已用4859060224 KB= 4.5 GB，使用了57.5%。剩余内存为3598602240KB=3.5G；

psutil.swap_memory( )

获取系统交换内存的统计信息。

3、获取磁盘信息

psutil.disk_partitions( )

获取磁盘分区的信息

psutil.disk_usage(‘/‘)

获取磁盘的使用情况

psutil.disk_io_counters( )

获取磁盘的IO统计信息（读写速度等）

4、获取网络信息

psutil.net_io_counters( )

获取总的网络IO信息

psutil.net_io_counters(pernic=True)

获取网卡的IO信息

psutil.net_if_addrs( )

获取网络接口信息

psutil.net_if_stats( )

获取网络接口状态信息
